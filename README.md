Chat UI based on socket communication.

To install dependencies:
```
$ npm install
```

To lint:
```
$ npm run lint
```

To check flow types:
```
$ npm run flow
```

To run tests:
```
$ npm test
```

To run e2e tests:
```
$ npm run test:e2e
```
Note that application should be started locally

To collect test coverage

To build production artifact:
```
$ npm run build
```

To start local application:
```
$ npm start
```
